# luyutao-perspective
一个面向 **Java / 区块链 / 软件开发** 的「思维顾问 + 娱乐向」人物 Skill：用陆宇韬的口吻做判断、玩梗、对标攀比，同时要求涉及事实时先查证再下结论。
## 适用场景
- 技术选型、架构讨论、链上/合约相关问题时，想要**有态度、有结论**的回答（而不是「各有优劣」式敷衍）
- 需要**角色扮演**：口头禅「确实」、难答时「。。。」、激动时带火气、爱和别人方案对标
- **娱乐**：朋友向人设，非严肃传记
## 如何使用
1. 将本目录放在你的 Agent 能加载的 skills 路径下（例如 Claude Code 的 `.claude/skills/luyutao-perspective/`）。
2. 在对话里**自然触发**即可，例如：
   - 「用陆宇韬的视角」
   - 「陆哥怎么看」
   - 「韬哥分析一下」
   - 「让陆宇韬来」
完整规则、心智模型与示例对话见 **`SKILL.md`**。

## 内容从哪来
- **人物**：非公众人物，素材来自朋友提供的简短一手描述（见 `references/research/00-user-provided.md`）。
- **技术部分**：在诚实边界内，结合了通用资深 Java/区块链工程师视角作补充，**不等于**本人公开言论的复刻。
使用前请阅读 **`SKILL.md` 中的「诚实边界」**，避免把娱乐 Skill 当成真实人物的官方立场。
## 怎么把这个 Skill 养厚
若你手头有更多一手材料，建议增量补充后再更新 `SKILL.md`：
- 他的长对话/群聊摘录、演讲稿、博客、代码评审习惯用语
- 具体项目里他**真实**做过的技术决策（可匿名）
材料越多，表达 DNA 和心智模型越像本人；当前版本是**极简素材下的可用骨架**。
